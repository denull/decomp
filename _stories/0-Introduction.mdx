# Decomp UI

A universal, themable component library for Svelte 5. Designed to look and feel native across platforms.

## Themes

Decomp ships with **6 visual themes** that closely imitate real system controls:

| Theme | Inspired by |
|---|---|
| **Semi-Flat** | Clean, modern default |
| **iOS / macOS** | Apple Human Interface Guidelines |
| **Android** | Material Design 3 / Material You |
| **Windows** | Fluent Design System |
| **Skeuomorphic** | Vintage analog hardware — cassette players, radios, DJ mixers |
| **Win98** | Classic Windows 98 |

Each theme supports **4 color schemes**: Day, Night, High Contrast Day, High Contrast Night.

## Getting Started

Wrap your app in `ThemeProvider`:

```svelte
<script>
  import { ThemeProvider } from '$lib';
</script>

<ThemeProvider theme="semiflat" scheme="day" accent="oklch(0.6 0.2 280)">
  <!-- your app -->
</ThemeProvider>
```

## Components

- **Primitives**: Button, Check (checkbox/radio/toggle), Slider
- **Inputs**: Input (text, multiline, search, tags, select, combobox)
- **Lists**: List, ListItem, Group
- **Navigation**: Tabs (top tabs + bottom tab bar)
- **Layout**: AppShell, Toolbar, Sidebar, Card, Stack, Divider
- **Overlays**: Overlay (popover, modal, tooltip)
- **Feedback**: Progress (bar + spinner)
- **Display**: Badge, Avatar

## Accent Color

Set a custom accent color via the `accent` prop on `ThemeProvider`:

```svelte
<ThemeProvider accent="oklch(0.6 0.25 330)">
```

All accent-derived colors (hover, active, subtle, text) are computed automatically using CSS `oklch(from ...)`.
